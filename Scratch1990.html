The 1990 Scratch Prototype - "+S+"

In the shadowy past of educational programming tools lies a forgotten prototype known only to a few obscure developers and digital historians. Before the Scratch we know today, there existed a strange, unreleased program called +S+, created sometime around 1990 by a secret collaboration between MIT Media Lab and Disney Interactive.

Origins and Platform

+S+ was never made public. It was designed to run on Windows 3.0 through 2000, but oddly, it also works on Windows 8.1 if launched from the cmd prompt. It refuses to run on Windows 10 or 11. It's incompatible with Windows Whistler and all Windows Server editions.

The Interface and The Shadow

+S+ didn’t have many features. Its most notable element was a bizarrely overcomplicated version of the Scratch Cat. The only known image of the program wasn’t a screenshot — it was a photograph taken of a computer monitor, and in the reflection of the screen, a mysterious shadow figure appears, standing just behind the person taking the photo.

Super Secret Settings

Users who tried to register at scratch-mit.disney.cc could access a hidden setting: by clicking the Scratch Cat icon (which glitches on hover), the actual program launches, but now the UI is corrupted. The screen fills with:

  • Distorted images of the Scratch Cat
  • Glitched screenshots of the early editor
  • Real photos of the MIT Media Lab and Disney HQ
  • Fragments of Mickey Mouse in bizarre, static-filled poses

After viewing the images, the computer crashes and becomes unresponsive. Antivirus software identifies +S+.exe as malware or a virus.

The Mystery of Æ

The file Æ.sb0 is believed to be the key. Some say something dark is hidden within — possibly code fragments, or maybe an embedded recording. MIT and Disney seemed obsessed with the "Æ" character, and it appears repeatedly throughout the prototype’s file system.

The 1991 Leak

In April 1991, +S+ was accidentally made public for a brief period. A 9-year-old child downloaded it and saved it on a floppy disk. That disk, along with other hard drives and storage, was lost to time. The now 44-year-old man still owns the damaged hardware, but all attempts to recover the file have failed. Some say he still dreams about the distorted cat and static-ridden Mickey.

Forgotten Ad on Disney DX

Between 1987 and 1989, a low-budget advertisement aired briefly on Disney DX. It mentioned a "revolutionary creativity tool" for kids called +S+. No footage remains, and Disney denies ever airing it.

Conclusion

The 1990 Scratch Prototype remains one of the internet’s most obscure and chilling digital mysteries. Whether a hoax, a lost experiment, or something else entirely, one thing’s certain — if you ever find a copy of +S+.exe or the file Æ.sb0, run. Or… maybe open it. Just don’t say we didn’t warn you.
